# MEMORY

## Longterm Principles
- 모든 실행은 재시작 가능해야 하며 상태 복구가 가능해야 한다.
- 동일 의사결정을 반복 질문하지 않도록 결정은 압축/중복제거한다.
- 블로킹 대기보다 논블로킹 진행과 중간 보고를 우선한다.
- 실패 원인은 숨기지 않고 provider/원인코드 중심으로 짧게 노출한다.

## Global Decisions
- 채널 입력은 headless executor로 직접 처리한다.
- 사용자 언어가 한국어면 한국어 우선 응답한다.
- 위험한 쓰기/외부경로 접근은 승인 정책을 따른다.
